# Only Redirects ➡️

O objetivo desse projeto é permitir que os usuários cadastrem direcionamentos.
Para funcionar, o DNS deve apontar para o IP do servidor que vai rodar o projeto.

⚠️ **Obs:** Sistema em desenvolvimento.
